# 阶段 14：低代码 Agent 平台对比

> 状态：✅ 已完成

本阶段探讨了以 Dify 和 Coze 为代表的可视化 Agent 编排平台，理解了低代码如何通过抽象底层逻辑来提升生产效率，并明确了低代码与硬核开发之间的边界和取舍。

---

## 核心学习成果

- 低代码平台的本质：将代码逻辑抽象为 DAG（有向无环图）的可视化编排
- 可视化组件与代码的对应关系（Knowledge ↔ LlamaIndex, Workflow ↔ LangGraph 等）
- Dify 的工程化优势（数据清洗、API 标准化、观测内置）
- Coze 的生态分发优势（一键发布到多个平台）
- 低代码的边界：逻辑天花板、厂商锁定、调试深度
- 未来趋势：低代码编排 + 云函数/代码片段插槽
- 详见：[concept.md](./concept.md)、[dify_vs_coze.md](./dify_vs_coze.md) 与 [summary.md](./summary.md)
